# GapR_ATvsSC
Jupyter Notebooks and assocaited processed data are used to generate figures in NAR manuscript: "GapR stiffens AT-rich and overtwisted DNA leading to differential effects on transcription based on gene context" 

First Notebook to run: Step1_downlaods_and_ChIP_analysis
This notebook downloads all necessary datasets from NCBI and builds a dataframe with peak calling and cahracterization to be saved into the ./data folder

Second Notebook to run: Fig8_S2
This notebook contains subsequent analysis, statistics and graphing for all panels of Fig 8/S2

Third notebook to run: Fig 9_S3_S4
This notebook contains subsequent analysis, statistics and graphing for all panels of Fig 9/S3/S4
